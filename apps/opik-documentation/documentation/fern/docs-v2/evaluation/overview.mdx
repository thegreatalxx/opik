## Two approaches to evaluation

{/* TODO: Brief overview of both, link to concepts for details */}

Opik provides two complementary approaches to evaluation:

- **Evaluation Suites**: Define natural-language assertions and let an LLM judge check them automatically. Best for pass/fail testing of specific behaviors.
- **Datasets & Metrics**: Score your agent's outputs against a dataset using pre-built or custom metrics. Best for measuring quality across many traces with quantitative scores.

## Key features

{/* TODO: Review and expand each bullet */}

- **Evaluation Suites** with natural-language assertions and execution policies

- **Annotation Queues** for human-in-the-loop review
