## Key features

{/* TODO: Review and expand each bullet */}

- **Traces and spans**: Capture the full execution path of every request, from input to output
- **Conversations**: Group related traces into threads to track multi-turn interactions
- **User feedback**: Log qualitative and quantitative feedback on traces
- **Media and attachments**: Log images, audio, video, and other files alongside your traces
- **Cost tracking**: Monitor token usage and spending across all your LLM calls
- **Integrations**: First-class support for 30+ frameworks in Python, TypeScript, and OpenTelemetry

## How it works

{/* TODO: Consider adding a diagram */}

1. **Instrument your code** — Add Opik tracing to your application using the SDK or an integration
2. **View traces** — Every request creates a trace with detailed span-level information
3. **Analyze and improve** — Use the Opik dashboard to debug issues, track costs, and monitor quality
