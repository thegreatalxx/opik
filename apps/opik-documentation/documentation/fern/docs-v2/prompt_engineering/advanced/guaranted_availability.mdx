Opik is a highly reliable platform and we dedicate significant effort to ensure that agent
configurations are consistently served. The SDK provides two layers of protection against
unavailability:

## Caching layer

The SDK automatically caches the last successfully fetched configuration in memory with a
default TTL of **300 seconds (5 minutes)**. A background thread proactively refreshes stale
cache entries before they expire, so your agent never blocks on a network call in steady state.

If Opik is unreachable during a background refresh, the SDK silently keeps serving the
last cached value — your agent continues running without interruption.

You can tune the cache TTL by setting the `OPIK_CONFIG_TTL_SECONDS` environment variable:

```bash
OPIK_CONFIG_TTL_SECONDS=60  # refresh every minute
```

## Fallback configuration

For the first call (cold start) or if the cache is empty and Opik is unreachable, you can
provide a hardcoded fallback configuration that the SDK will use instead of raising an error:

<CodeBlocks>
  ```python title="Python"
  import opik

  client = opik.Opik()

  # Define the schema for my configuration
  class MyAgentsConfiguration(opik.Config):
      model: str = "gpt-4o-mini"
      temperature: float = 0.75
      system_prompt: opik.Prompt = None

  FALLBACK_CONFIGURATION = MyAgentsConfiguration(
    model="gpt-4o-mini",
    temperature=0.75,
    system_prompt=opik.Prompt(name="system_prompt", prompt="You are an AI assistant.")
  )

  cfg = client.get_or_create_config(
    fallback=FALLBACK_CONFIGURATION,
    project_name="my-agent-name",
    env="latest",
  )

  print(cfg.model)
  print(cfg.temperature)
  print(cfg.system_prompt)
  ```

  ```ts title="Typescript"
  import Opik, { Prompt } from "opik";

  const client = new Opik();

  const fallbackPrompt = await client.createPrompt({
    name: "system_prompt",
    prompt: "You are an AI assistant.",
  });

  const cfg = await client.getOrCreateConfig({
    projectName: "my-agent-name",
    env: "latest",
    fallback: {
      model: "gpt-4o-mini",
      temperature: 0.75,
      system_prompt: fallbackPrompt,
    }
  });

  console.log(cfg.model);
  console.log(cfg.temperature);
  console.log(cfg.system_prompt);
  ```
</CodeBlocks>
